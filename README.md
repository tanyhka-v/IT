# GitHub User Finder

**Автор:** Зарвинова Татьяна

## Описание
GUI-приложение для поиска пользователей GitHub с использованием официального API. Позволяет просматривать аватары, добавлять пользователей в избранное и сохранять список в JSON-файл.
