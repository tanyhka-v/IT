# Expense Tracker

**Автор:** Татьяна. (tanyhka-v)

## Описание
Приложение для учёта личных расходов с графическим интерфейсом (Tkinter). Позволяет добавлять расходы с суммой, категорией и датой; фильтровать по категории и диапазону дат; удалять записи; сохранять все данные в JSON-файл.

## Установка и запуск
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/tanyhka-v/IT.git
   cd IУбедитесь, что у вас установлен Python 3.6+.

Запустите приложение:
python expense_tracker.py

   Как использовать
Добавление расхода: введите сумму (положительное число), выберите категорию, укажите дату в формате YYYY-MM-DD, нажмите "Add Expense".

Фильтрация: выберите категорию (или "All") и/или диапазон дат, нажмите "Apply Filter". Таблица обновится, и внизу появится итоговая сумма за период.

Удаление: выделите строку в таблице и нажмите "Delete Selected".

Сохранение: все изменения автоматически сохраняются в expenses.json (кнопка "Save to JSON" дублирует сохранение вручную).

Валидация
Сумма должна быть положительным числом.

Дата должна соответствовать формату YYYY-MM-DD.

Поле суммы не может быть пустым.

Примеры использования
Добавить расход: сумма 12.50, категория Food, дата 2025-05-12.

Отфильтровать расходы за май 2025: from 2025-05-01, to 2025-05-31.

Посмотреть общую сумму трат на транспорт.
