Запуск

1. Убедитесь, что установлен Python 3.7+ и библиотеки:
      pip install requests pillow
2. Запустите файл main.py:
      python main.py

Поиск пользователей

· Введите логин (или его часть) в текстовое поле.
· Нажмите Поиск или клавишу Enter.
· Результаты появятся в таблице: логин, аватар (иконка), ссылка на профиль.

Избранное

· Нажмите ☆ Добавить рядом с пользователем – он сохранится в favorites.json.
· Текст кнопки изменится на ★ Удалить – нажмите, чтобы убрать из избранного.
· Чтобы просмотреть только избранных – нажмите кнопку Показать избранное.

Открытие профиля

· Дважды кликните на любой строке таблицы – откроется профиль GitHub в браузере.

Важно

· Поле поиска не должно быть пустым (появится предупреждение).
· Избранное сохраняется автоматически и восстанавливается при следующем запуске.
· При проблемах с интернетом или API GitHub – вы увидите сообщение об ошибке.
